# Calculadora Mobile

## Sobre o projeto
Este projeto foi criado para fins de estudo.

O objetivo principal e praticar as linguagens e tecnologias usadas na aplicacao, alem de ajudar na ambientizacao com desenvolvimento front-end.

## Objetivos de estudo
- Praticar JavaScript e React Native.
- Entender melhor a estrutura de um app com Expo.
- Treinar criacao de interfaces e componentes no front-end.
- Evoluir boas praticas de organizacao de codigo.

## Observacao
Este projeto tem foco educacional e de aprendizado continuo.
